# TrendingShoes - Premium Footwear E-commerce Website

Welcome to **TrendingShoes**, a modern and responsive e-commerce website dedicated to premium footwear for men, women, and kids. This project is built with **React** and styled using **Tailwind CSS** to provide a sleek and user-friendly shopping experience.

## Features

- Responsive landing page showcasing featured products and categories
- Dedicated pages for Men’s shoes, Women’s shoes, Kids’ shoes, and Contact Us
- Dynamic product listing using React components and data mapping
- Contact form with validation
- Newsletter subscription section
- Clean and modern UI with Tailwind CSS
- Font Awesome icons for enhanced visuals

---

## Technologies Used

- [React](https://reactjs.org/) - Frontend library
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [Font Awesome](https://fontawesome.com/) - Icon library
- HTML5 & CSS3
- JavaScript (ES6+)
